Habilidades e Interesses

HTML, CSS e JavaScript

HTML, CSS e Javascript, também conhecidas como santíssima trindade da web, formam o principal conjunto de tecnologias para qualquer aplicação web. Já deu para ver que é de suma importância entender MUITO bem esses caras.

Python

Python é uma linguagem de programação versátil e poderosa, amplamente reconhecida por sua simplicidade e legibilidade. Com sua sintaxe intuitiva e vasta biblioteca padrão, Python é uma escolha popular para uma ampla gama de aplicações, desde desenvolvimento web até ciência de dados e automação.

Postgre e SQL

SQL, a Linguagem de Consulta Estruturada, é a base para interagir com bancos de dados relacionais. O PostgreSQL, é um poderoso sistema de gerenciamento de banco de dados relacional de código aberto que utiliza o SQL como sua linguagem de consulta padrão. O SQL permite a criação, consulta, atualização e exclusão de dados em um banco de dados. Com o PostgreSQL, essas operações são estendidas para incluir recursos avançados, como suporte a tipos de dados personalizados, procedimentos armazenados, funções definidas pelo usuário e gatilhos.
